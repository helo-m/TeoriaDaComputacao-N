# Documentação
## Operações implementadas
---
> `mov(A, B, C)`

A = B (usando C)

> `sum(A, B, C)`

A = A + B (usando C)

> `sub(A, B, C)`

A = A - B (usando C)

> `clr(A)`

A = 0

> `mul(A, B, C, D)`

A = A * B (usando C e D)

> `mul2(A, B, C)`

A = 2 * A (usando C)

> `exp2_(A, B, C)` _-> not implemented_

A = 2^A (usando B e C)

> `exp2(A, B, C, D)`

B = 2^A (usando C e D)



## testes implementados
---
> `both_zero(A, B)`

A == 0 && B == 0

> `cmpAB(A, B, C)`

A == B (usando C)
